Lockin Digitizer.lvclass:Actor Core.vi

(Module: Lockin Digitizer)

Digitizer records pulsed waveforms on external trigger signal. Digitizer is cofigured via Digitizer config part of the Configuration cluster. Optional Lock-in synthesis and detection is configurev via Lock-in config part of the cluster.

Digitizer is armed by the main module Arm buttom . If Armed light is bright green, it is ready for the trigger. After pulse is triggered, Armed goes dark. After pulse is recorded, Done turns bright red.
